Agentic Development Training

Your handover — read this before you start.

This is a self-paced training program covering everything from your first deliberate use of an AI coding assistant through to shipping production agents with evals, observability, and rollback plans. You'll work through one short module per week, then have a 15-minute check-in with your engineer lead. About 90 minutes a week total. Spread it however suits you.

Get started in 5 steps

  1. Clone the repo and open a terminal in it.
    git clone <repo-url> && cd agentic-dev-training
  2. Install dependencies (one time).
    npm install
  3. Launch the local app.
    npm run dev — opens in your browser at localhost:5173.
  4. Take the placement quiz (10 questions, ~5 minutes). The app sets up your cohort folder automatically with your name, score, and starting level.
  5. Open your first module from the My modules page and follow the read / do / self-check flow. When you're done, reach out to your engineer lead for a 15-minute check-in.
Don't want to use the app? Open module files directly from modules/ in your editor and read placement-quiz.md to find your starting level.

The four levels

LevelYou are here if…Capstone
1 · Novice You've used AI tools casually. No daily, deliberate workflow yet. Ship a real PR with Claude Code, plus a 600-word retro.
2 · Practitioner Confident daily user. Have called the API at least once. Build a small internal tool with tool use + a 10-case eval suite.
3 · Proficient Have shipped an LLM feature. Hit at least one production surprise. Ship a production agent with CI evals, observability, safety, rollback plan.
4 · Expert Architect, mentor, standard-setter. None — sustaining role.

How a week looks

~75 min

Self-paced module

Read the linked docs (~15 min), do the hands-on exercise with starter code (~60 min), run the self-check.

~15 min

Check-in with the lead

Reach out via Teams or however your team talks. Bring the artefacts the module lists. The lead probes 2–3 questions and signs you off.

Real work

Capstone

Each level's capstone is applied to a real piece of project work — not a side project. Demo it to the lead when ready.

What you need before day 1

  • Python 3.10+ on your machine.
  • An Anthropic API key — set as the ANTHROPIC_API_KEY environment variable. Ask the engineer lead if you don't have one.
  • Git access to the training repo (clone, push to your own cohort folder).
  • Claude Code installed for the L1 modules (docs at docs.claude.com).
